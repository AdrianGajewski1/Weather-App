# Weather-App
A application that shows current weather based on city, city id or geographic coordinates. 

# Technologies and frameworks used:
  * Asp net Core 3.1 
  * Angular 9 
  * OpenWeatherMap API 
  * Angular Material UI
  
# Requirements
  * Dotnet SDK
  * Npm CLI
  
# Usage
  * WebAPI
      * On the project root directory run
      ```
      dotnet build
      ```
       then
      ```
      dotnet run
      ```
  * Angular frontend
     * On the project src directory run
      ```
      npm start
      ```
